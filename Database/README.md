# Database - 设计数据和资产

此目录包含设计数据和资产。

## 子目录说明

| 目录 | 用途 |
|------|------|
| [Docs](./Docs/) | 按技术领域分类的设计文档 |
| [DesignData](./DesignData/) | RTL、网表、版图数据 |
| [Verification](./Verification/) | 验证环境、用例、报告 |
| [Validation](./Validation/) | FPGA和硅后验证 |
| [Firmware](./Firmware/) | 固件代码和驱动 |
| [Scripts](./Scripts/) | EDA工具流程脚本 |
| [Reference](./Reference/) | IP手册、协议规范 |
| [ToolConfig](./ToolConfig/) | 工具配置文件 |

## 文件命名规范

```
设计文档:     [Module]_[Type]_Spec_v[Version]_YYYYMMDD.md
脚本:         run_[Flow]_v[Version].tcl
报告:         [Type]_Report_[Date].pdf
```
