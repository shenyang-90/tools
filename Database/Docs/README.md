# Docs - 设计文档

此目录按技术领域分类的设计文档。

## 子目录

| 目录 | 用途 |
|------|------|
| [Arch](./Arch/) | 架构文档 |
| [Design](./Design/) | 设计文档 (模块规格、接口规格) |
| [Verification](./Verification/) | 验证文档 |
| [FuSa](./FuSa/) | 功能安全文档 |
| [Firmware](./Firmware/) | 固件文档 |
| [DFT](./DFT/) | DFT文档 |
| [Physical](./Physical/) | 物理设计文档 |

## 文件命名规范

```
[Module]_[Type]_Spec_v[Version]_YYYYMMDD.md

示例:
- CPU_Subsystem_Design_Spec_v1.0_20260327.md
- DDR_Controller_Interface_Spec_v1.2_20260415.md
```
