# MeetingMinutes - 日常会议记录

此目录包含日常会议记录。

## 会议类型

| 类型 | 频率 | 参与人 |
|------|------|--------|
| 项目例会 | 每周 | PM + 各Lead |
| 架构评审 | 按需 | 全体架构师 |
| RTL评审 | 每两周 | Design + Verification |
| 验证同步 | 每周 | Verification + Design |
| 后端评审 | 每周 | PD + DFT + STA |
| 安全评审 | 每月 | FuSa + 相关方 |

## 文件命名规范

```
Meeting_Minutes_YYYYMMDD_[Topic].md

示例:
- Meeting_Minutes_20260327_Weekly_Review.md
- Meeting_Minutes_20260328_Architecture_Discussion.md
```
