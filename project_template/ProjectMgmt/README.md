# ProjectMgmt - 项目管理

此目录包含项目管理层面的文档和记录。

## 子目录说明

| 目录 | 用途 |
|------|------|
| [Planning](./Planning/) | 项目计划、Schedule、资源规划 |
| [Reviews](./Reviews/) | 各阶段评审记录 (PCD/PAD/EDR/IDR/FDR/PostSilicon) |
| [Bugs](./Bugs/) | Bug分类管理 (P1/P2/P3/P4) |
| [MeetingMinutes](./MeetingMinutes/) | 日常会议记录 |
| [Milestones](./Milestones/) | 阶段交付物归档 |
| [RiskMgmt](./RiskMgmt/) | 风险评估与跟踪 |
| [ChangeMgmt](./ChangeMgmt/) | ECO/CR记录 |
| [StatusReports](./StatusReports/) | 项目状态报告 (周报月报) |

## 文件命名规范

```
会议记录:     Meeting_Minutes_YYYYMMDD_[Topic].md
评审记录:     [Phase]_Review_Meeting_Minutes_YYYYMMDD.md
Bug记录:      BUG_[ID]_[Brief_Description].md
风险登记:     RISK_[ID]_[Description].md
变更请求:     CR_[ID]_[Description].md
状态报告:     Weekly_Status_YYYY_W[Week].md
              Monthly_Status_YYYY-MM.md
```
