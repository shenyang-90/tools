# Bugs - Bug分类管理

此目录按严重程度分类管理Bug。

## 分类说明

| 目录 | 严重级别 | 说明 | 响应时间 |
|------|---------|------|---------|
| [P1_Critical](./P1_Critical/) | P1 - 致命 | 系统崩溃、数据丢失、安全风险 | 24小时内 |
| [P2_Major](./P2_Major/) | P2 - 严重 | 主要功能失效、性能严重下降 | 3天内 |
| [P3_Minor](./P3_Minor/) | P3 - 一般 | 次要功能问题、轻微性能问题 | 1周内 |
| [P4_Trivial](./P4_Trivial/) | P4 - 轻微 | 界面问题、文档错误 | 2周内 |

## Bug状态流转

```
Open → Assigned → InProgress → Fixed → Verified → Closed
  ↑                                              ↓
  └────────────── Reopen ←───────────────────────┘
```

## 文件命名规范

```
BUG_[ID]_[Brief_Description].md

示例:
- BUG_001_Clock_Domain_Crossing.md
- BUG_002_Memory_Leak.md
```
